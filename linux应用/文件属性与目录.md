linux中用stat可以查看文件信息，本质上是调用`stat()`函数
```cpp
#include <sys/types.h>
#include <sys/stat.h>
#include <unistd.h>

int stat(const char *pathname, struct stat *buf);
```
第二个参数buf指针，会存获取的文件信息
返回值：成功返回 0；失败返回-1，并设置error
