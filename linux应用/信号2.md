## sigaction
`sigaction(int signum,const struct sigaction *act,struct sigaction *oldact)`
- 第一个参数是需要设置的信号
- 第二个参数是一个sigaction结构体指针
- 第三个参数也是一个sigaction结构体指针，如果不设置成NULL，会把这个信号之前的处理方式通过oldact这个结构体返回出来
 #### sigaction结构体
 ![[Pasted image 20251222223112.png]]
 ![[Pasted image 20251222223127.png]]
```c
static void my_handler(int sig)
{
	printf("recieved signal!",sig)
}
int main(int argc,char *argv[])
{
int ret;
struct sigaction m_sigaction={0};
m_sigaction.sa_handler=my_handler;
m_sigaction.flags=0;
ret =sigaction(SIG_INT,&sigaction,NULL)
if(ret==1)
	perror("error")
}
```

## kill
向指定进程发送信号
`ret=kill(pid,信号名)`
成功返回0，失败返回-1、

## raise
向本线程发送信号

## alarm
alarm（time）
设置为0，就是取消之前的闹钟
如果一开始有定时，那返回值就是上次定时的剩余时间
定时结束的时候，会发送SIG_ALRM

## puase
执行后
直到进程捕获一个信号之前，休眠

## abort
用来异常终止进程
会向进程发送SIG_ABRT
