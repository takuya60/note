file oprations这个结构体里面是一大堆函数指针
对应了我们在应用层调用的时候的函数
比如在应用层
```c
int fd;
// 这里打开的是设备节点
open("dev/m_dev",fd,O_RDWR);
```
这时候，就会调用fops结构体里面的open函数
```c
struct file_oprations m_fops(){
	.owner=THIS_MOUDLE,
	.open=m_dev_open ,
	.read=m_dev_close,
	.......
}
```
具体m_dev_open怎么写
直接ctrl 点进file_oprations的定义，去里面复制
