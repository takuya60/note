手动创建的方法是mknod
这里只讲自动创建
需要先自己在sys/class里面创建一个新类和一个新设备
mdev这个程序会自动跟据sys/class新建的类和设备，自动生成dev/xxx
```c
struct class *m_class;
struct device *m_device;

// 在init里面
m_class =class_creat(THIS_MOUDLE ,"m_dev_class")
m_device= device_creat(m_class,NULL,dev_id,NULL,"my_dev")
// 这样就会生成 sys/class/m_dev_class/my_dev

// 在exit里面
device_destory(m_device,m_class);
class_destory(m_class);
```
