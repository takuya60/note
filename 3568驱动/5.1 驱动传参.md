## 驱动传参可以传三种
## 1.普通类型（int char....）
## 2.数组
## 3.string
分别对应Linux里的三个函数 在include/linux/moudleparam.h里定义
## 1.
```c
#include <linux/moudleparam.h>
static int a;
moudle_param(a ,int ,S_IRUGO);
// 第一个参数是参数名字，第二个是类型，第三个是访问权限，一般用444
```
## 2.
```c
static int arr[6]={0};
static int arr_size;
moudle_param_array(arr ,int ,&arr_size ,S_IRUGO );
// 第一个参数是参数名字，第二个是类型，第三个是访问权限，一般用444
```
## 3.
```c
static char str[6]={0};
moudle_param_string(str ,str ,sizeof(str) ,S_IRUGO );
// 第一个参数是参数名字，第二个是类型，第三个是访问权限，一般用444
```
