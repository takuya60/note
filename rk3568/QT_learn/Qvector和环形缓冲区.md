# Qvector
## 特点
- **连续内存 (Contiguous Memory)**：这是它最重要的特性。不管你往里面存多少个数据（比如 `int` 或结构体），它们在内存条里都是肩并肩挨在一起的，中间没有缝隙。
- **动态扩展**：你不需要像定义普通数组那样（`int array[10]`）把大小写死。用 `QVector`，你想装多少就装多少，不够了它会自动去申请更大的地盘。
## 强弱项
- 强项：访问极快
- 弱项：头部操作极慢，如果在头部插入，后面的都要往后移动
```cpp
Qvector<int> scores
//添加数据(快)
socres.append(95);
scores<<96<<97; //简便写法，可以一下写入两个数据
//在头部添加（慢）
scores.prepend(100);
```
# 环形缓冲区
当头部写入时，不往后移动后面的数据，
而是把写入位置调整到头部，覆盖前面的数据
有一个`write_index`（写入位置），指向下一个要放数据的地方
假设我们的缓冲区大小固定是 `10,000` (索引范围 0~9999)。当你的 `write_index` 已经写到了最末尾的 **`9999`**，而新的数据又来了，我们需要把`write_index`调回0
```cpp
// 当 m_writeIndex 为 9999 时： 如果再写入，index自动变成0
// % 取余
 m_writeIndex = (m_writeIndex + 1) % BUFFER_SIZE;
```
