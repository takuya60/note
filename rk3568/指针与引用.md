
# 在linux中用write写入
需要`write(fd,写入的指针/地址,大小)`
所以如果要写入结构体
```cpp
struct A packet
write(fd,&packet,sizeof(packet));
```
# 强制类型转换

在 C++ 中，为了安全，类型检查非常严格。如果你在使用一些旧的 API，或者需要对指针进行加减运算（比如 `ptr + 1` 移动一个字节），你就必须把它看作 `char*`
所以可以把结构体地址强制转换为char*
```cpp
write(fd, reinterpret_cast<char*>(&packet), sizeof(packet));
```
