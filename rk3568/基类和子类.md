

`IBackend`是基类
`RPMsgBackend`和`MocBackend`是子类
在写子类的时候需要
```cpp
class RSMgsBackend : IBackend {
}
```
在基类中，对函数进行声明 可以用纯虚函数
```cpp
virtual void updateParameters(const StimulationSettings &settings) = 0;
```
在子类.h中，对函数进行声明
```cpp
void updateParameters(const StimulationSettings &settings) override;
//一定要写override
```
在子类.cpp中，对函数进行定义和实现
```cpp
// 具体功能的实现 
void RPMsgBackend::startStimulation(const StimulationSettings &settings) { 
// 调用内部打包函数 
packAndSend(CMD_START, settings); }
```
在main函数里
```cpp
Ibackend *backend =nullptr；
# ifdef Q_OS_LINUX
	// 如果是在板子上，new 一个真机后端
	bakend = new RPMsgBackend();
# else 
	// 如果是在 Windows 电脑上，new 一个模拟后端
	backend = new MockBackend();
#endif

Treatmanager *manager = new Treatmanager(backend);
```
